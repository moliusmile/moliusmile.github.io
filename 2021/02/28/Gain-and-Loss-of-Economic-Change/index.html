《历代经济变革得失》 吴晓波

author Shuxin Yang time 2021-02-28
book 历史
《历代经济变革得失》 吴晓波

我们所见的事实及所作出的论断，总是受到时代视角、意义语境的局限。

版本信息

  • 出版社：浙江大学出版社
  • 出版时间：2016年11月
  • 荐书人：在北大读了四年历史的仔西同学

导论 研究中国的方法

【内容】

  1. “人们可以用三种方式去研究经济：通过理论、通过统计和通过历史。”于是，回到“中国历史的基本面”，从历代经济变革中探研得失，寻找规律与逻辑，也许是一次不错的探险。
  2. 统一是中国的一种文化。
  3. 两个研究工具：
    • 四大利益集团博弈法：中央政府、地方政府、有产阶级和无产阶级
    • 中央集权与四大基本制度：郡县制度、尊儒制度、科举制度、国有专营制度

【感想】

  • “关于政治经济、经济发展、经济时代问题，都要理解决策者背后的思想模式和意识形态。所谓的意识形态，就是一群人对环境的解释，以及对该环境如何调理出秩序，所提出对策背后一套共有的思想模式。”

第一讲 管仲变法：两千多年前的“凯恩斯主义”

【内容】

  1. 四民分业，士农工商：把国民分成军士、农民、工匠、商贾四个阶层，按各自专业聚居在固定的地区。
  2. 放活微观：对内刺激商品经济的发育，对外降低关税，形成“如水归壑”的市场聚集效应。
  3. 管制宏观：强调政府对经济的宏观管理，其手段则是从财政、税收和价格三方面综合入手。
  4. 盐铁专营：由政府控制资源所有权，然后把经营权下放给民间商人，以一定比例分配利润，是后世非常流行的“资产国有、承包经营”的雏形。
  5. 以商止战：
    • 就国家内战而言，“以商止战”就是发展商品经济，让国民富裕而不至于造反。
    • 就与各诸侯国的关系而言，“以商止战”就是扩大对外贸易，并以军事的威慑力维持均衡。

【感想】

  • 管仲的变法涉及许多经济命题，是一位尊重市场规律的国家干预主义者，是中国古代版的“凯恩斯”。

第二讲 商鞅变法：命令型计划经济的鼻祖

【内容】

  1. 以农立国：废井田，开阡陌
  2. 军爵制度：抹杀贵族和贱民的界限，人人可以通过战争获取功名富贵
  3. 郡县制度：全面加强中央集权，是地方行政制度的政治雏形
  4. 在商鞅的经济思想中，“强国”与“富民”似乎是对立的。他极端地认为，人民不应该有思考的能力，而且绝对不能够富足。

【感想】

  • 商鞅的施政手段暴烈，后世的名声非常差，“惠王车裂之，而秦人不怜”；但他的变法真正达到了强盛国家和统一天下的目标，被证明是有效果的和成功的，后世遵其法。

第三讲 汉武帝变法：顶层设计的集大成者

【内容】

  1. 汉武帝推出一系列国营化经济政策，具体操盘人是桑弘羊。
    • 产业改革：铸钱、盐铁与酿酒
    • 流通改革：均输（统购统销）与平准（物价管制）
    • 税收改革：告缗令与算缗令
  2. 盐铁会议：在桑弘羊看来，儒生来自民间，都没有治国的经验，只能提出国营化的弊端，却提不出有建设性的意见。提出了“桑弘羊之问”。

【感想】

  • “桑弘羊之问”在当下也具有很强的现实意义，面对社会问题，很多人只会提出问题抱怨现状而没有有效的解决方案。我觉得与其这样，还不如多想想如何乐观应对。

第四讲 王莽变法：第一个社会主义者的改革

【内容】

  1. 变法背景：世族经济威胁中央集权
  2. 王莽改革具体措施
    • 财政：五均六筦
    • 货币：七年四次币制改革
    • 土地：恢复全面的土地国有制，然后平均分配给农民耕种
  3. 王莽改革失败的原因
    • 经济集权的前提是政治集权，王莽以外戚身份取汉而代之，政权的合法性问题没有得到解决。
    • 中央集权改革将侵蚀地方政权和有产阶层的利益，所以改革成功的前提是，务必获得底层民众的支持。王莽变法未得民心。
    • 民间财富不够雄厚，并且握于世族集团之手搜刮难度大，因此中央政府无法获得更多的经济权益。
    • 王莽用人多疑，好换将帅，没有一支衷心高效的执行团队。

【感想】

  • 王莽执政的时间虽短，不过搞的经济改革却轰轰烈烈。
  • 但一场大的社会变革如同空间重构，疏处应可跑马，密处必不容针，王莽变法缺乏系统思考和风险预警，最终仍失败告终。

第五讲 世民治国：最盛的王朝与最小的政府

【内容】

  1. 推行科举制：造成知识阶层对国家权力的绝对依赖；也是对世族模式的一次彻底“反动”。
  2. 解决吏政、兵政之患：让富人出钱养官，让地方出钱养兵，中央政府的财政支出就变得很少，“小政府”因此而生。
  3. 轻税简政，促进工商业和地方经济的发展，城市经济空前繁荣。
  4. 中唐之后民间工商资本的五条出路：奢侈消费、交通权贵、购买土地、放高利贷、囤积钱币。

【感想】

  • 中晚唐民间资本的这五条出路几乎是高压下民间资本的共同出路。若在某一时期，出现奢侈品消费剧增、文物价格上涨以及土地房产购买热潮，并不代表经济的复苏，而更可能是资本从实业溢出的噩兆。

第六讲 王安石变法：最后的整体配套改革

【内容】

  1. 宋代制度
    • 政治制度：“杯酒释兵权”，中央养兵
    • 经济制度：国有专营制度更严酷；公开允许官员经商；不抑制土地兼并；对垄断资源进行授权经营。
  2. 王安石改革
    • 对经济权力的重组
    • 法令：均输法、市易法、青苗法、农田水利法、免役法和方田均税法

【感想】

  • 青苗法的初衷是好的，但是一到执行阶段就完全变味了。给我的启发是一件事能不能办好，不仅要有好的idea，好的方案，还要有好的执行和落地。
  • 王安石变法的同一时期，西方出现了“自由城市”和人类历史上第一所大学。自由的城市催生自由的商业，自由的大学催生自由的思想，而自由的商业和思想又是人类文明走向现代社会的两块奠基石。11世纪至12世纪常被称为中西方文明的大分流时期。

第七讲 明清停滞：大陆孤立主义的后果

【内容】

  1. 明清治国者思想变得保守的原因：对制度创新的畏惧。
  2. 铁桶阵和散沙术–明清专制的法宝：政权要稳定，危险无非来自两处，一曰外患，一曰内忧。除外患，断绝一切联系是不二法门，所以要把国家变成一个铁桶。解内忧，控制、削弱民间的组织力量是关键，所以要把人民打成散沙。
  3. 明清商人对官场的依赖：三大商帮（徽商、晋商、陕商）尽管都富可敌国，可都是被豢养大的寄生虫，他们的财富增加与市场的充分竞争无关，与产业开拓无关，与技术革新无关，因而与进步无关。

【感想】

  • 明清商人没有培育出“商人精神”，根本原因是，从知识精英到他们自己，都不认同商人是一个独立的阶层。如费正清所言，“中国商人最大的成功是，他们的子孙不再是商人”。这种商人对官场的依赖，在当下中国也仍是普遍存在的。

第八讲 洋务运动：缺乏现代性的现代化变革

【内容】

  1. 发展与稳定的取舍：修铁路，主流意识是反对。认为经济发展的终极目的，不是繁荣商贸，而是维持均衡，稳定政权，稳定永远被置于发展之上。
  2. 国营与民营的取舍：轮船招商局，重归官僚官制。
  3. 中日对比：在百年现代化的历程中，中国与日本有三次站在相同的“成长线”上。
    • 第一次是19世纪70年代，两国几乎同时分别开始了洋务运动和明治维新；
    • 第二次是1945年，日本战败，中国抗战结束，两国俱受到巨大的战争创伤；
    • 第二次便是当前，两国经济总量相当，分列全球第二和第三。
  4. 洋务运动制度建设角度的四条缺陷：
    • 缺乏制度上的顶层设计
    • 从未涉及财政、货币和土地政策
    • 不是一个普惠性的经济振兴运动
    • 现代化有余，现代性不足

【感想】

  • 中国的现代化并不是一场“迟到的运动”，迟到不是落后的理由。

第九讲 两个民国：从极度自由到“统制经济”

【内容】

  1. 1911~1927年的北洋民国：
    • 民营经济空前强大，最重要的事件是金融业的民营化
    • 地方自治的试验：军阀自治，商人自治，建设模范城市或实验公社
  2. 1927~1949年的国民党民国：
    • 经济主张：国家主义，推行优先发展国营企业的战略
    • 挤压民间资本：组织瓦解；国债绑架；控制证券市场；利用经济危机，大量收编民营企业；银行国有化。

【感想】

  • 国民党政府在敌产国营化中失去了私营企业家的支持，在货币改革中失去了城市居民的支持，而在土地改革中失去了农民的支持。

第十讲 计划经济：从自负到自毁的大试验

【内容】

  1. 1949年，上海试验：共产党人在流通和资本管制上获得了经验。
  2. 1953年开始，“第一个五年计划”：计划经济模式在工业建设上的一次完美表现。
  3. 1948年开始，七年消灭私营经济
  4. 1958年，两次自毁性的经济运动：“大跃进”，工业方面的目标是超过英国，农业方面的目标是实现粮食和棉花产量一年翻一番。
  5. 1958年，农民手中的土地收归为集体所有，政府获得三大收益
    • 第一个收益是控制粮食交易，实现农业对工业的“反哺”。
    • 第二个收益是推行票证经济，在物资短缺条件下长期控制城市物价。
    • 第三个收益是控制人口流动，进而消化城市剩余劳动力。
  6. 1966~1976，“文化大革命”紊乱十年

【感想】

  • 到1976年，中国已是一个封闭自守的、与世界经济体系基本“绝缘”、高度集中而没有活力的经济体。

第十一讲 改革开放（上）：没有蓝图的改革

【内容】

  1. 由农民发动的工业化运动：农村改革中，乡镇企业异军突起。
  2. 中央政府的角色
    • 放权让利：对象是国营企业和地方政府，目标是刺激经济要素的复苏和减少中央财政支出。
    • 非均衡战略：特区、开发区模式是资源配置上的一次非均衡；“东南优先”是区域经济发展的一次非均衡；“让一部分人先富起来”是财富分配的一次非均衡；给予外资以土地征用和税收上的“超国民待遇”是企业经营上的非均衡；企业试点是政策配置上的非均衡；价格双轨制则是对国营企业实施价格保护的非均衡。
  3. 地方政府的角色
    • 积极参与：以集体企业为主力的苏南模式
    • 冒险支持：以私营企业为主力的温州模式
  4. 哪里是改革的主战场
    • 企业主体改革派，厉以宁
    • 整体协调改革派，吴敬琏

【感想】

  • 波兰人布鲁斯：“经济体制改革要以政治体制改革为条件，必须从制度上采取措施，才能保证不再回到旧轨道上去。归根结底，改革是政治和社会问题。”

第十二讲 改革开放（下）：集权主义的回归

【内容】

  1. 整体改革措施
    • 建立新的财政税收体制：分税制
    • 启动金融——银行体系的改革
    • 进行外汇管理体制改革
    • 推进国有企业的改革
    • 建立新的社会保障制度
  2. 1988年的“三驾马车”
    • 启动城市化建设
    • 开放外贸的进出口自主权
    • 刺激内需，开放房地产市场
  3. 在经历了2004年的宏观调控后，国有资本与民营资本的产业界限已然划定：前者在资源、能源性领域取得了垄断性的地位，而后者则被压缩在中下游的产业领域，如历史上一再出现的景象，两者楚河汉界，泾渭分明。

【感想】

  • 任何制度如同胚胎，一旦形成便会惯性生长，具有自我实现的能力，若没有良好的制衡性机制，其最终的形态甚至会超出设计者的初衷和预期。

结语：回到历史的基本面

【内容】

  1. 一个前提：“统一文化”是一切自由化改革的边界
  2. 两个主题：分权与均富
    • 分权，主要指的是两类分权，即中央政府与地方政府的权益分配、政府与民间的权益分配。若没有中央对地方的分权，政府对民间的分权便不可能发生。
    • 均富，也分为两类，即政府与民间的均富，以及有产者与无产者的均富，其内在关系是，若没有政府对民间的均富，有产者与无产者之间的均富也不可能发生。
  3. 三个战场：国有经济、土地和金融业
  4. 四股新势力：互联网、非政府组织、企业家和自由知识分子

【感想】

  • 我们所见的事实及所作出的论断，总是受到时代视角、意义语境的局限。
