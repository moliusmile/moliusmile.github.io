《TED演讲的秘密》

《TED演讲的秘密》

你也许永远没有机会真正地登上TED舞台，但你依然需要用激励他人的方式去传递自己的想法。

版本信息

  • 出版社：中国人民大学出版社
  • 出版时间：2014年4月
  • 荐书人：上官老师

引言

【内容】

  1. TED代表着Technology、Entertainment和Design。
  2. TED的使命是分享并推广有价值有意义的创意灵感。
  3. 你也许永远没有机会真正地登上TED舞台，但你依然需要用激励他人的方式去传递自己的想法。

【感想】

  • 我想学习如何演讲的契机，一是为我之后可能的登台演讲做准备；二是提升自己传递想法的能力。
  • 在人与人链接的社会，如何更好地沟通，如何传播自己的思想，是至关重要的。

01 确定主题：明确你要“传播的观点”和你的“角色定位”

【内容】

  1. 演讲角色“定位”：教育者，娱乐者，变革推动者
  2. 选择话题时要进行深层次的以结果为导向的反思。
  3. 演讲者常犯的一个最大的错误就是试图通过一次演讲传达一生所学，其实，专注于一个概念可以让你更清晰地编辑你的材料。
  4. 讨论你“熟悉与热爱”的话题。
  5. 演讲的目的是“给予”，出场是为了分享我所知道的东西。

【感想】

  • 演讲首先要专注于一个话题，目的是分享我的观点。

02 构思讲稿：观点是脊柱，论点是肋骨，论据是血肉

【内容】

  1. 尽管TED以讲述精彩的故事著称，但事实上，大部分演讲者都是以道理为主，然后用多个故事作为论据支撑，丰富演讲内容。通常，讲故事的演讲者会用第一人称展开叙述。
  2. 逻辑在演讲中的应用：归纳推理（观点在开头或结尾），演绎推理（观点常在结尾）
  3. 兼具论点和论据的5要素演讲提纲
    导论->A部分论据->A部分论点->B部分论据->B部分论点->C部分论据->C部分论点->结论
  4. 除了故事外的其他形式：统计、引用、举例、假设

【感想】

  • 演讲不仅要以故事感染听众，其内在也需要有一定的逻辑主线。

03 编排故事：“英雄之旅”是最基础的故事讲述模式

【内容】

  1. 讲述“亲身经历”或“亲自观察”到的故事：一堂课；决定性的时刻；克服弱点
  2. 使用“英雄之旅”的模式：正常世界-冒险召唤-拒斥召唤-见导师-越过第一道边界-考验、伙伴、敌人-接近最深的洞穴-磨难-报酬-返回的路-复活-携万能药回归
  3. 伟大故事的标志，是通过诠释让听众逐层领会其中的智慧。
  4. 通过“外貌与对话”让人物变得鲜活：拥有弱点和复杂人性的真实人物才是所有引人入胜故事的基础。

【感想】

  • 快乐是比痛苦更有效的长效激励；使用积极的故事远比使用消极的故事要成功。
  • 通过拥抱人类的多样性，我们将找到获得真正幸福的可靠方法。

04 把握关键：口号、开场白、过渡与结尾

【内容】

  1. 将观点提炼为口号：控制在3-12个词；以行动为中心；富有韵律；至少重复3次
  2. 最引人入胜的三种开场方式：
    • 以“有针对性的故事”开场
    • 以“震撼人心的事实”开场
    • 以“有影响力的问题”开场
  3. 过渡：好的过渡能精准地回顾前一部分，同时能巧妙地预告下一部分。它还能给观众提供宝贵的时间，将刚才所讲的内容同演讲的整体逻辑联系起来，并同个人经历联系起来。
  4. 结尾：一种有效的表明结尾的方法是缩短你的句子，在声音中加入激情。用“你，我们，大家”来代替“我”。

【感想】

  • 内容才是王道。目标应该是成为一个能演讲的专家，而不是一个专业演说家。

05 调动情绪：语言技巧与非语言技巧完美配合

【内容】

  1. 语言技巧：
    • 语气热情谦逊（语言表现出一种极具感染力的好奇、探索欲和敬畏心）
    • 绝对不要使用口头禅
    • 发挥停顿的力量
    • 改变语速、音量和音调。打造有变化的声音
  2. 非语言技巧
    • 双手自然下垂
    • 双腿采用模特站姿
    • 手势一定要有，但不能使用过度
    • 表情配合内容，眼神与观众交流（真诚微笑；眼神接触）

【感想】

  • 当演讲者走上讲台，就要把自己变成观众最亲密无间的朋友，而不是表现得过于专业或聪慧。因此要使用日常用语，也要能在台上释放最愉悦和最脆弱的时刻。

06 善用工具：用与不用的艺术

【内容】

  1. 幻灯片：尽量不用。要有不用幻灯片演讲的能力。除非两个例外：
    • 演讲者有能够极大充实演讲内容的个人照片；
    • 演讲者需要展示数据
  2. 视频：尽量不用。如果要用需要注意以下几点：
    • 视频长度最好在30秒以内
    • 站在一侧与听众一起观看视频
    • 对无声视频进行解说
  3. 讲台：尽量不用。除非三个理由：
    • 演讲者需要刻意地展现力量和权威
    • 由活动或观众的规格而定的
    • 演讲者需要依靠笔记

【感想】

  • 工具只是辅助演讲，核心还是在于演讲本身，不能让工具令观众分心。即使要用也要注意“越少越好”的原则。

07 其他准备：优秀的演讲要重视每一个细节

【内容】

  1. 穿着：穿着舒适，作真实的自己，符合自己的职业
  2. 活出自己的热情，真实地信奉一个值得传播的观点
  3. 练习：练习的目标是实现对话式演讲，避免死记硬背或是直接读稿；列出演讲提纲，而不是逐字稿。
  4. 介绍词：介绍演讲对于听众的益处；将主题相关度扩展到最大；将履历资料减至最少。
  5. “病毒式传播”的视频特征：出人意料、创新、情感诉求

【感想】

  • 看演讲相关的书以及视频并不能成为优秀的演讲者，唯一可行的方法就是克服恐惧，在各种演讲场合不断打磨自己的演讲技巧。
